# casbin 权限认证管理

> 一个权限认证的库，主要是用于Go语言的，也实现了其他语言的扩展。我也是看到[奇淼](https://github.com/piexlmax)的B站关于这个的介绍，所以才接触到这个
所以想把这引入到FastAPI中。

## 安装依赖
```
pip install casbin
pip install casbin_sqlalchemy_adapter
```

## 学前准备

可以先在 [官方编辑器](https://casbin.org/zh-CN/editor)里面弄明白不同访问策略的原理。
